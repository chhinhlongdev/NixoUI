# Roblox Script Development Guide

This guide helps you communicate with me effectively to create, modify, or fix Roblox scripts.

---

## 📝 Quick Command Templates

### Creating New Scripts

**Simple Script:**
```
create new simple script with [features you want]
```

**Game-Specific Script:**
```
create script for [game name] with [features]
```

**Example:**
```
create new simple script with fly, speed, and ESP
create script for Blox Fruits with auto farm and teleport
```

---

### Adding Features

**Add Single Feature:**
```
add [feature name] to [script name]
```

**Add Multiple Features:**
```
add [feature1], [feature2], and [feature3] to [script name]
```

**Example:**
```
add ESP to Simple_Script.lua
add fly speed slider, noclip, and teleport to player to Blox_Fruit.lua
```

---

### Fixing Issues

**Fix Specific Problem:**
```
fix [problem description] in [script name]
```

**Fix Visual/UI Issues:**
```
fix scroll/layout/button/animation in [script name]
```

**Example:**
```
fix the fly not working in Simple_Script.lua
fix scroll and smooth animations in Blox_Fruit.lua
```

---

### Upgrading Features

**Upgrade Existing Feature:**
```
upgrade [feature name] in [script name]
```

**Add More Options:**
```
add more options to [feature name] in [script name]
```

**Example:**
```
upgrade ESP in Blox_Fruit.lua
add more options to movement tab in Simple_Script.lua
```

## 🎯 Common Features You Can Request

### Movement Features
- `fly` - Flying with WASD controls
- `fly speed slider` - Adjustable fly speed (0-100%)
- `walk speed` - Adjustable walk speed
- `jump power` - Adjustable jump height
- `infinite jump` - Jump infinitely in air
- `noclip` - Walk through walls
- `teleport` - Teleport to locations/players

### Visual Features
- `ESP players` - See players through walls
- `ESP enemies` - See enemies through walls
- `ESP items` - See items through walls
- `ESP chests` - See chests through walls
- `ESP fruits` - See devil fruits through walls
- `ESP tracers` - Lines pointing to targets
- `ESP boxes` - 3D boxes around targets
- `ESP distance` - Show distance to targets
- `ESP health` - Show health bars
- `full bright` - Remove darkness

### Player Interaction
- `teleport to player` - TP to selected player
- `kill player` - Eliminate selected player
- `bring player` - Teleport player to you
- `player dropdown` - Select player from list
- `spectate player` - Watch selected player

### Farming Features (Game-Specific)
- `auto farm` - Automatic enemy farming
- `auto quest` - Automatic quest completion
- `auto collect` - Collect items automatically
- `fast attack` - Rapid attack speed
- `weapon selection` - Choose weapon type
- `skill usage` - Auto-use abilities

### GUI Features
- `minimize button` - Minimize to icon
- `close button` - Close/destroy GUI
- `draggable` - Move window around
- `toggle keybind` - Show/hide with key
- `smooth animations` - Animated transitions
- `tabs` - Organize features in tabs
- `sliders` - Adjustable value controls
- `toggles` - On/off switches
- `dropdowns` - Selection menus

### Utility Features
- `anti-afk` - Prevent AFK kick
- `auto respawn` - Respawn automatically
- `remove fog` - Clear fog effects
- `god mode` - Invincibility (if possible)
- `infinite stamina` - Never run out of stamina
